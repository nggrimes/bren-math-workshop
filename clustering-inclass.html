Now we can add some settings on how big we want the groups to be or how many groups we want. For example if out a class of 80 we want group sizes of 4 we would need to tell the algorithm to make 20 groups. Since there might be an uneven number (like 83) weâ€™ll tweak the max allowed in a group. For balanced sorting all groups must have at least one representative from each group. So the minimum size will be a rounding of the number of responses.

## mix up function

mix<-function(data,group){
  df_filter<-data %>% 
      filter(labels==group)
    
  out=sample(df_filter$name,size=length(df_filter$labels))
  
  return(out)
}
